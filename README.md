This project is a Spring application.  
The main purpose is to develop a facebook app with the help of Spring-Social.

Project is created with:
* Java 8
* Spring Boot 2.3.4.  
* MySQL
* Maven  
* AngularJS v1.2.32  
* jQuery  
* Bootstrap  
IDE: Intellij IDEA 2020
